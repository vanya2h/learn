## Highlights

This release renames the product from "Learning Tracker" to **Sheafu** across all user-facing surfaces — page titles, headers, footers, auth pages, and the README. It also migrates all route-level imports from relative paths to `~/` TypeScript path aliases, cleaning up the import style throughout the app.

## Changes

### New Features

- **Path aliases**: All route files under `app/routes/` now use the `~/` alias instead of deep relative imports (e.g. `../../src/components/...`). Affects all route files and hooks (`useLocale.ts`, `useRootData.ts`, `root.tsx`). — `packages/web/app/routes/`, `packages/web/app/hooks/`

### Bug Fixes

- Sign-up form name placeholder changed from `Ivan K.` to `John Doe` to avoid a locale-specific example — `packages/web/app/routes/sign-up.tsx`
